# AbfrageGenerator

Zentrale Sammlung von SQL-Abfragen für CSR-Software-Kunden.

Diese Abfragen können direkt in den Access-Abfrage-Generator eingefügt
oder per URL automatisch geladen werden.

## Ordnerstruktur
- `/sql` – reine SQL-Dateien
- `/catalog` – index.json Datei (Abfrage-Katalog für Access)
- `/pages` – HTML-Seiten mit Kopier-Button
